◆ kDof

template<class TScalar , int kDim>
constexpr int sophus::UnitVector< TScalar, kDim >::kDof = kDim - 1
staticconstexpr

The DOF is one dimension smaller than the embedding dimension. Examples:

  • A unit circle lives in a 2d space, but a point on a circle can be represented using a single scalar (e.g. an angle).
  • A unit sphere lives in a 3d space, but a point on a sphere can be described using a 2-tuple (e.g. latitude and longitude).
